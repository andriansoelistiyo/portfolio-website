- **Build Tool**: Vite 6.3.5

- **Icons**: Lucide React
- **Typography**: Playfair Display (headings) + Libre Franklin (body)

## Sections

1. **Hero** - Introduction with contact info
2. **About** - Professional summary and background
3. **Professional Experience** - Work history with show more/less functionality
4. **Personal Projects** - Featured projects and case studies
5. **Skills** - Technical skills and tools expertise
6. **Contact** - Direct contact information

## Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npx vite preview
```

## Deployment

This project is configured for easy deployment to Netlify:

1. Push to GitHub
2. Connect repository to Netlify
3. Netlify will auto-detect build settings from `netlify.toml`
4. Deploy!
